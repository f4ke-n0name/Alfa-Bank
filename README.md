# Bookshelf

## Запуск
Лабораторная реализована как command line tool проект. Сделано это, чтобы упростить запуск и демонстрацию.
Запуск происходит в main.swift → создаётся экземпляр CLI, в который нужно передать объект, соотвествующий протоколу BookShelfProtocol, и вызывается run().

## Реализованные команды

-  add     — добавить книгу
-  delete  — удалить книгу по ID
-  list    — показать все книги
-  search  — поиск по названию/автору/жанру/тегам
-  help    — справка
-  exit    — выход

### Пример работы
```text
=== BookShelf CLI ===
Доступные команды: add, list, search, delete, help, exit

Введите команду:
add

--- Добавление книги ---
Название:
1984
Автор: 
George Orwell
Год издания:
1949
Жанры (fiction, nonFiction, mystery, sciFi, biography, fantasy) через запятую:
sciFi
Теги через запятую:
uk, classic
Книга добавлена: 1984
```
